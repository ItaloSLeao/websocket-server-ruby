// core concepts
01
TCP Foundation
WebSocket rides on top of TCP — the transport layer guarantees ordered, reliable byte delivery. The OS handles the three-way handshake (SYN → SYN-ACK → ACK) before our code even runs.
02
HTTP Upgrade
Every WebSocket starts as a plain HTTP GET request. The client sends Upgrade: websocket. The server responds with 101 Switching Protocols. After that line break — HTTP is gone forever.
03
SHA-1 Handshake
The client sends a random base64 key. The server concatenates it with the RFC 6455 magic string, SHA-1 hashes the result, and returns it as Sec-WebSocket-Accept. Browser validates before proceeding.
04
Binary Frames
Post-handshake, HTTP text is replaced by binary frames. Each frame is parsed byte-by-byte: FIN bit, opcode (text/binary/close/ping), mask flag, payload length, masking key, and finally the payload.
05
XOR Unmasking
The RFC requires all client→server frames to be masked. Each payload byte is XOR'd with the corresponding byte of a 4-byte key (cycling with % 4). XOR is reversible: (A⊕B)⊕B = A.
06
Full Duplex
Unlike HTTP's request→response→close cycle, the WebSocket connection stays open. Either side can send frames at any time — enabling real-time chat, live dashboards, and multiplayer games.
// connection lifecycle
01
TCP_CONNECT
Browser opens a TCP connection to localhost:2345. OS performs SYN → SYN-ACK → ACK three-way handshake.
02
HTTP_UPGRADE_REQUEST
Browser sends GET / with headers: Upgrade: websocket, Sec-WebSocket-Key: [random base64]
03
SHA1_HANDSHAKE
Server computes SHA1(key + magic_string), base64-encodes it, replies with HTTP 101 + Sec-WebSocket-Accept.
04
FRAME_PARSE
Browser sends binary frame. Server reads byte-by-byte: FIN/opcode → mask/length → masking key → payload.
05
XOR_UNMASK
Server XORs each payload byte against mask[i % 4], converting masked bytes back to readable UTF-8.
06
FRAME_RESPONSE
Server builds response frame [0b10000001, size, payload] and writes it. Browser fires onmessage callback.
// the unmasking algorithm
# XOR unmasking — RFC 6455 §5.3
mask = 4.times.map { socket.getbyte }
data = payload_size.times.map { socket.getbyte }

unmasked = data.each_with_index.map do |byte, i|
  byte ^ mask[i % 4]  # XOR is reversible: (A⊕B)⊕B = A
end

message = unmasked.pack('C*').force_encoding('utf-8')
