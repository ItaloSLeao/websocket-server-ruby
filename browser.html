// core concepts
01
TCP Foundation
WebSocket rides on top of TCP. The OS handles the three-way handshake (SYN → SYN-ACK → ACK) before our code runs. Ordered, reliable byte delivery is guaranteed.
02
HTTP Upgrade
Every WebSocket starts as a plain HTTP GET with Upgrade: websocket. The server responds with 101 Switching Protocols. After the blank line — HTTP is gone forever.
03
SHA-1 Handshake
The client sends a random base64 key. The server concatenates it with the RFC 6455 magic string, SHA-1 hashes the result, and returns it as Sec-WebSocket-Accept.
04
Binary Frames
Post-handshake, HTTP text is replaced by binary frames parsed byte-by-byte: FIN bit, opcode, mask flag, payload length, masking key, payload.
05
XOR Unmasking
All client→server frames are masked per RFC 6455. Each payload byte is XOR'd with the corresponding byte of a 4-byte key (cycling with % 4). XOR is reversible: (A⊕B)⊕B = A.
06
Full Duplex
Unlike HTTP's request→response→close, the WebSocket connection stays open. Either side sends frames at any time — enabling chat, live dashboards, and multiplayer games.
// connection lifecycle
01
TCP_CONNECT
Browser opens TCP connection to localhost:2345. OS performs SYN → SYN-ACK → ACK.
02
HTTP_UPGRADE_REQUEST
Browser sends GET / with Upgrade: websocket and Sec-WebSocket-Key: [random base64].
03
SHA1_HANDSHAKE
Server computes SHA1(key + magic_string), base64-encodes it, replies with HTTP 101.
04
FRAME_PARSE
Server reads frames byte-by-byte in a loop: FIN/opcode → mask/length → masking key → payload.
05
XOR_UNMASK
Server XORs each payload byte against mask[i % 4], converting masked bytes to UTF-8.
06
FRAME_RESPONSE
Server builds response frame [0b10000001, bytesize, payload] and writes it. Browser fires onmessage.
// the unmasking algorithm
# XOR unmasking — RFC 6455 §5.3
mask = 4.times.map { socket.getbyte }
data = payload_size.times.map { socket.getbyte }

unmasked = data.each_with_index.map do |byte, i|
  byte ^ mask[i % 4]  # XOR is reversible: (A⊕B)⊕B = A
end

message = unmasked.pack('C*').force_encoding('utf-8')
